    <p>Após a confirmação de três casos de gripe aviária no <a href="https://tribunaonline.com.br/tags/es?home=esp%C3%ADrito+santo&amp;d=1" target="_blank">Espírito Santo</a>, a Secretaria de Estado da Saúde (Sesa) enviou amostras biológicas de 33 pessoas que, possivelmente, tiveram contato com as aves infectadas pelo vírus H5N1. Em comunicado divulgado na tarde desta quarta-feira (17), a pasta informa que essas pessoas podem ter tido contato com os animais no <a href="https://tribunaonline.com.br/cidades/visitas-sao-suspensas-em-parque-de-jardim-camburi-apos-caso-de-gripe-aviaria-140147?home=esp%C3%ADrito+santo" target="_blank">Parque Fazendinha, localizado no bairro Jardim Camburi, em Vitória</a>, e um dos locais onde as aves doentes foram encontradas.&nbsp;</p> 
    
    
    
    
    
    
    

    <p>De acordo com a Sesa, uma dessas pessoas apresenta sintomas gripais leves e está em isolamento, sendo monitorada por profissionais da Prefeitura de Vitória.&nbsp;</p> 
    
        <div id="banner-video-area"></div>
        
            <div class="Super-Leaderboard-Centro-Mobile-ES" style="display: none; text-align: center;" id="div-gpt-ad-1730822898962-0"></div>
            <div class="Super-Leaderboard-Centro-Mobile-PE" style="display: none; text-align: center;" id="div-gpt-ad-1730823322355-0"></div>
        
    
    
    
    
    
    
    

    <p><a href="https://tribunaonline.com.br/cidades?home=esp%C3%ADrito+santo&amp;d=1" target="_blank">Leia mais notícias de Cidades aqui</a></p> 
    
    
        <div id="banner-video-area"></div>
        
            <div class="Super-Leaderboard-Centro-Desktop-ES" style="display: none; text-align: center;" id="div-gpt-ad-1730823135349-0"></div>

    <p>A investigação de possíveis expostos às aves contaminadas começou a ser feita na terça-feira (16)&nbsp;pelas equipes da Vigilância em Saúde da Sesa, juntamente com o Instituto de Defesa Agropecuária e Florestal do Espírito Santo (Idaf), nas cidades de Vitória, Cariacica e Marataízes.&nbsp;</p> 
    
    
    
        
            <div class="Super-Leaderboard-A-Mobile-ES" style="display: none; text-align: center;" id="div-gpt-ad-1730824244605-0"></div>
            <div class="Super-Leaderboard-A-Mobile-PE" style="display: none; text-align: center;" id="div-gpt-ad-1730824521079-0"></div>
        
    
    
    
    
    

    <p>"Na manhã desta quarta-feira (17), o município de Vitória realizou a coleta das 33 amostras de contatos e enviou ao Laboratório Central de Saúde Pública do Espírito Santo (Lacen-ES). Após conferência do material, o Lacen enviará para análise na Fundação Oswaldo Cruz (Fiocruz) nesta quinta-feira (18)", informou a Sesa.&nbsp;</p> 
    
    
    
    
    
    
    

    <p>Ainda não há informação sobre o prazo para divulgação dos resultados dessas análises.<br></p> 
    
    
    
    
        
            <div class="Super-Leaderboard-A-Desktop-ES" style="display: none; text-align: center;" id="div-gpt-ad-1730824400055-0"></div>
            <div class="Super-Leaderboard-A-Desktop-PE" style="display: none; text-align: center;" id="div-gpt-ad-1730824601679-0"></div>
            <div class="Super-Leaderboard-B-Mobile-ES" style="display: none; text-align: center;" id="div-gpt-ad-1730825281747-0"></div>
            <div class="Super-Leaderboard-B-Mobile-PE" style="display: none; text-align: center;" id="div-gpt-ad-1730825541281-0"></div>
        
    
    
    
    

    <p>A Fiocruz é o laboratório de referência para área de abrangência nacional que engloba o Lacen-ES, uma vez que as amostras deverão ser manipuladas em áreas de biossegurança NB-3.&nbsp;</p> 
    
    
    
    
    
    
    

    <p>A Sesa esclarece que as pessoas que tiveram contato com as aves diagnosticadas com a gripe aviária (vírus H5N1) devem ser monitoradas se apresentarem sintomas gripais pelo período de dez dias a partir do contato com a ave contaminada ou com suspeita de contaminação.&nbsp;&nbsp;</p> 
    
    
    
    
    
        
            <div class="Super-Leaderboard-C-Mobile-ES" style="display: none; text-align: center;" id="div-gpt-ad-1730825774701-0"></div>
            <div class="Super-Leaderboard-C-Mobile-PE" style="display: none; text-align: center;" id="div-gpt-ad-1730825935850-0"></div>
        
    
    
    

    <h3 style="color: rgb(0, 0, 0);"><b>LEIA MAIS</b></h3> 
    
    
    
    
    
    
        
            <div class="Super-Leaderboard-B-Desktop-ES" style="display: none; text-align: center;" id="div-gpt-ad-1730825196783-0"></div>
            <div class="Super-Leaderboard-B-Desktop-PE" style="display: none; text-align: center;" id="div-gpt-ad-1730825469366-0"></div>
        
    
    

    <p><a href="https://tribunaonline.com.br/economia/ministro-descarta-risco-de-restricao-comercial-apos-casos-de-gripe-aviaria-140142?home=esp%C3%ADrito+santo" target="_blank">Ministro descarta risco de restrição comercial após casos de gripe aviária</a></p> 
    
    
    
    
    
    
    

    <p><a href="https://tribunaonline.com.br/brasil/casos-de-gripe-aviaria-registrados-no-es-sao-os-primeiros-do-brasil-em-2023-140095?home=esp%C3%ADrito+santo" target="_blank">Casos de gripe aviária registrados no ES são os primeiros do Brasil em 2023</a></p> 
    
    
    
    
    
    
    

    <h2><b>Casos confirmados no Espírito Santo</b></h2> 
    
    
    
    
    
    
    
        
            <div class="Super-Leaderboard-C-Desktop-ES" style="display: none; text-align: center;" id="div-gpt-ad-1730825662200-0"></div>
            <div class="Super-Leaderboard-C-Desktop-PE" style="display: none; text-align: center;" id="div-gpt-ad-1730825852947-0"></div>
        
    

    <p><a href="https://tribunaonline.com.br/saude/dois-casos-de-gripe-aviaria-sao-confirmados-no-es-140085?home=esp%C3%ADrito+santo" target="_blank">Duas aves marinhas e migratórias da espécie T</a><i><a href="https://tribunaonline.com.br/saude/dois-casos-de-gripe-aviaria-sao-confirmados-no-es-140085?home=esp%C3%ADrito+santo" target="_blank">halasseus acuflavidus</a></i><a href="https://tribunaonline.com.br/saude/dois-casos-de-gripe-aviaria-sao-confirmados-no-es-140085?home=esp%C3%ADrito+santo" target="_blank"> (Trinta-réis-de-bando) foram recolhidas entre 7 e 8 de maio. Uma no litoral de Marataízes, no Sul do Estado, e outra em Jardim Camburi, em Vitória.</a></p> 
    
    
    
    
    
    
    

    <p>Elas estavam debilitadas e, por isso, encaminhadas ao Instituto de Pesquisa e Reabilitação de Animais Marinhos de Cariacica (Ipram).&nbsp;</p> 
    
    
    
    
    
    
    

    <p>Na última quarta-feira (10), após as aves apresentaram sinais clínicos neurológicos, o instituto notificou a suspeita ao Instituto de Defesa Agropecuária e Florestal do Estado (Idaf), que coletou amostras para análise.</p> 
    
    
    
    
    
    
    

    <p>Também foram detectados sintomas em uma terceira ave migratória da espécie <i>Sula leucogaster</i> (atobá-pardo), que se encontrava no Ipram.</p> 
    
    
    
    
    
    
    

    <p>Os casos foram notificados ao Ministério da Agricultura e Pecuária (Mapa). Segundo a pasta, a confirmação de casos de influenza aviária em aves silvestres não afeta a condição do Brasil como país livre da doença.&nbsp;</p> 
    
    
    
    
    
    
    

    <p>Segundo o governo, não há risco ao consumo de carne e ovos, devido aos rígidos protocolos de biosseguridade do sistema comercial.&nbsp;</p> 
    
    
    
    
    
    
    

    <p>Em coletiva de imprensa na terça-feira (16),&nbsp;o secretário de Estado da Agricultura, <a href="https://tribunaonline.com.br/saude/gripe-aviaria-nao-ha-risco-em-consumir-ovos-e-aves-diz-secretario-do-es-140259?home=esp%C3%ADrito+santo" target="_blank">Enio Bergoli, também garantiu que o consumo de ovos e carne de frango é seguro. “Não há nenhum risco no consumo dos produtos”</a>.</p> 
    
    
    
    
    
    
    

    <h2><b>Orientações</b></h2> 
    
    
    
    
    
    
    

    <p>A pasta ainda reforça que a população deve evitar estritamente contato com aves doentes ou mortas, incluindo aves silvestres. Outras orientações gerais incluem:</p> 
    
    
    
    
    
    
    

    <p>- Ao avistar aves doentes acionar o serviço veterinário local ou realizar a notificação por meio do Sisbravet - <a href="https://sistemasweb.agricultura.gov.br/pages/SISBRAVET.html" target="_blank">https://sistemasweb.agricultura.gov.br/pages/SISBRAVET.html</a>;&nbsp;</p> 
    
    
    
    
    
    
    

    <p>- Evitar o contato próximo e desprotegido com pessoas que apresentem sintomas gripais;</p> 
    
    
    
    
    
    
    

    <p>- Manter os ambientes bem ventilados com porta e janelas abertas;</p> 
    
    
    
    
    
    
    

    <p>- Evitar aglomerações em ambientes fechados;</p> 
    
    
    
    
    
    
    

    <p>- Praticar higiene das mãos e ter etiqueta respiratória ao tossir ou espirrar;</p> 
    
    
    
    
    
    
    

    <p>- Caso precise manipular a ave, usar máscara de proteção e luvas, acondicionar em uma caixa e comunicar ao Idaf - <a href="https://idaf.es.gov.br/" target="_blank">www.idaf.es.gov.br</a>.</p>

        function setCookie(name, value, days) {
            var expires = "";
            if (days) {
                var date = new Date();
                date.setTime(date.getTime() + (days * 24 * 60 * 60 * 1000));
                expires = "; expires=" + date.toUTCString();

            }
            document.cookie = name + "=" + (value || "") + expires + "; path=/";
        }

        function getCookie(name) {
            var nameEQ = name + "=";
            var ca = document.cookie.split(';');
            for (var i = 0; i < ca.length; i++) {
                var c = ca[i];
                while (c.charAt(0) == ' ') c = c.substring(1, c.length);
                if (c.indexOf(nameEQ) == 0) return c.substring(nameEQ.length, c.length);
            }
            return null;
        }

        function ctrlEvent(e) {
            if (e.key === 'Control' || e.metaKey) {
                ctrlDown = true;
            }

            if (ctrlDown && e.key === 'c') {
                handleCopy();
            }
        }

        function handleCopy() {
            const selection = window.getSelection().toString();
            const urlAtual = window.location.href;
            const textoAdicional = `\nVeja mais em: ${urlAtual}`;
            const textoCopiado = selection + textoAdicional;

            navigator.clipboard.writeText(textoCopiado).then(function () {
                return;
            }, function (err) {
                console.error('Async: Could not copy text: ', err);
            });
        }
