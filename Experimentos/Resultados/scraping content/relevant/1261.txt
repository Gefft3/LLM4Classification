O tratamento, segundo anuncio de cientistas nesta terça-feira (15) promete curar pessoas de diversas origens raciais.&nbsp;</p> 
    
    
    
    
    
    
    

    <p>A&nbsp;prática comum é&nbsp; utilizar doadores de medula óssea de raça e etnia semelhantes à do paciente, que na maior parte dos registros é de origem caucasiana. No entanto, de acordo com a nova técnica é possível realizar o procedimento apenas com&nbsp; uma correspondência parcial, o que representa o potencial de curar dezenas de americanos que têm HIV e câncer a cada ano, disseram os cientistas. O sangue do cordão umbilical está mais amplamente disponível do que as células-tronco adultas, normalmente usadas em transplantes de medula óssea, e não precisa ser compatível com o receptor.&nbsp;</p> 
    
    
    
    
    
    
    

    <p>No caso da paciente, que também tinha leucemia,&nbsp; o sangue veio de um doador parcialmente compatível. A mulher também recebeu sangue de um parente próximo para dar ao seu corpo defesas imunológicas temporárias enquanto o transplante acontecia.O sexo e a origem racial do novo caso marcam um passo significativo no desenvolvimento de uma cura para o HIV, disseram os pesquisadores.</p>

    <p>"O fato de ela ser mestiça e ser mulher é muito importante cientificamente e muito importante em termos de impacto na comunidade. Estas são histórias de inspiração para o campo e talvez um caminho", disse Steven Deeks, especialista em Aids da Universidade da Califórnia, em São Francisco, que não estava envolvido no trabalho.</p>

    <p>Especialistas acreditam que&nbsp; infecção por HIV&nbsp; progrida de maneira diferente nas mulheres do que nos homens, no entanto, embora as mulheres representem mais da metade dos casos de HIV no mundo, elas representam apenas 11% dos participantes em estudos que buscam a cura.</p>

    <p>Drogas antirretrovirais poderosas podem controlar o HIV, mas a cura é a chave para acabar com a pandemia de décadas. Em todo o mundo, quase 38 milhões de pessoas vivem com HIV, e cerca de 73% delas estão recebendo tratamento.</p>

        function setCookie(name, value, days) {
            var expires = "";
            if (days) {
                var date = new Date();
                date.setTime(date.getTime() + (days * 24 * 60 * 60 * 1000));
                expires = "; expires=" + date.toUTCString();
                colsole.log('expira no dia: ' + date)
            }
            document.cookie = name + "=" + (value || "") + expires + "; path=/";
        }

        function getCookie(name) {
            var nameEQ = name + "=";
            var ca = document.cookie.split(';');
            for (var i = 0; i < ca.length; i++) {
                var c = ca[i];
                while (c.charAt(0) == ' ') c = c.substring(1, c.length);
                if (c.indexOf(nameEQ) == 0) return c.substring(nameEQ.length, c.length);
            }
            return null;
        }

        function eraseCookie(name) {

        let ctrlDown = false;

        function ctrlEvent(e) {
            if (e.key === 'Control' || e.metaKey) {
                ctrlDown = true;
            }

            if (ctrlDown && e.key === 'c') {
                handleCopy();
            }
        }

        function handleCopy() {
            const selection = window.getSelection().toString();
            const urlAtual = window.location.href;
            const textoAdicional = `\nVeja mais em: ${urlAtual}`;
            const textoCopiado = selection + textoAdicional;

            navigator.clipboard.writeText(textoCopiado).then(function () {
                return;
            }, function (err) {
                console.error('Async: Could not copy text: ', err);
            });
        }

        document.addEventListener('copy', function (e) {
            const selection = window.getSelection().toString();

            const textoCopiado = selection + textoAdicional;

            e.preventDefault();
            
            e.clipboardData.setData('text/plain', textoCopiado);
        });
