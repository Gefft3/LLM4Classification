<p>El <a href="https://www.elcaribe.com.do/panorama/salud/salud-publica-descarta-colera-en-defunciones-ocurridas-en-la-cienaga-barahona/" target="_blank" rel="noreferrer noopener">Ministerio de Salud Pública</a> reportó que 33 personas han dado positivo a cólera en la provincia de Barahona, de estos 29 son de nacionalidad haitiana y cuatro son dominicanos.</p>



<p>La<a href="https://map.gob.do/DirectorioFuncionarios/Consulta/InformacionFuncionario?funcionarioId=19754" target="_blank" rel="noreferrer noopener"> directora de Gestión de Riesgo de la institución, Gina Estrella a</a>firmó que desde que se originó el brote diarreico, en el Hospital Regional Universitario Jaime Mota de esta provincia sureña, ingresaron 12 dominicanos y 33 haitianos con evacuaciones diarreicas y de estos las estadísticas arriba descritas son los que han dado positivos.</p>



<p>Durante la acostumbrada rueda de prensa que ofrece el ministerio de salud Estrella declaró que de los cuatros ríos que presentaron contaminación, sólo en el río Ciénega persiste la presencia de Vibrio cólera.</p><div id="" class="mmc_ads "><style> #div-gpt-ad-elCaribe_300x250_inread>div iframe, #div-gpt-ad-elCaribe_300x250_inread >div{

<p>De igual manera, la directora de Gestión de Riesgo dijo que entre las acciones que se llevan a cabo contra el brote de cólera se ha intervenido seis sectores, se han aplicado 8,404 vacunas contra el cólera, al tiempo que se han hecho 3,859 tratamientos profilaxis doxiciclinas.</p><div id="" class="mmc_ads "><style> #div-gpt-ad-elCaribe_300x250_inread>div iframe, #div-gpt-ad-elCaribe_300x250_inread >div{

	function getCookie(cname) {

        let decodedCookie = decodeURIComponent(document.cookie);
        let ca = decodedCookie.split(';');
        for (let i = 0; i < ca.length; i++) {
            let c = ca[i];
            while (c.charAt(0) == ' ') {
                c = c.substring(1);
            }
            if (c.indexOf(name) == 0) {
                return c.substring(name.length, c.length);
            }
        }
        return "";
    }
