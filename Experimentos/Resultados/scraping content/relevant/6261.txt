<div class="accessibility-buttons">

                            </div><p style="text-align: justify;"><span style="font-weight: 400;">A <a href="https://veja.abril.com.br/ciencia/entender-agua-doce-e-importante-para-vida-na-terra-diz-geografo-americano" target="_blank" rel="noopener"><strong>água doce</strong></a> é um dos principais elementos para a sobrevivência humana na Terra, necessária não apenas para o consumo, mas também para a manutenção da natureza. O uso irracional deste recurso, no entanto, tem um preço. De acordo com um relatório divulgado nesta segunda-feira, 16, pelo Fundo Mundial para a Natureza (WWF), <strong>57,8 trilhões de dólares</strong> estão ameaçados pela degradação da água e dos ecossistemas de água doce.&nbsp;</span></p>
<p style="text-align: justify;"><span style="font-weight: 400;">O valor, equivalente a 60% do PIB global, foi calculado com base em perdas diretas e indiretas provocadas pela destruição desse recurso. As diretas, como uso na industrias, nas residências, nas hidrelétricas e na agricultura somam 7,5 trilhões. Os outros 50 trilhões são compostos por prejuízos indiretos, como saúde do solo, biodiversidade e eventos extremos como secas e inundações.&nbsp;</span></p>
<div class="ads video-ads">
<div class="abrAD" data-format="intext_video"></div>
</div>
<p style="text-align: justify;"><span style="font-weight: 400;">Esses riscos já se refletem em números. Desde 1970, um terço das <strong><a href="https://veja.abril.com.br/ciencia/mapa-mundial-da-agua-identifica-pontos-criticos-de-disponibilidade-hidrica" target="_blank" rel="noopener">áreas úmidas do planeta foram perdidas</a></strong> e as populações dos animais selvagens de água doce caíram 83%. No Brasil, a seca que assola a região norte já afeta grande parte dos corpos d’água, enquanto o <a href="https://veja.abril.com.br/coluna/matheus-leitao/marina-alerta-que-efeito-do-el-nino-na-amazonia-pode-se-estender-ate-2025/" target="_blank" rel="noopener"><strong>rio Amazonas</strong></a>, o mais extenso do mundo, tem sua biodiversidade ameaçada pelos rejeitos da mineração ilegal.&nbsp;</span></p>
<p style="text-align: justify;"><span style="font-weight: 400;">Para que essa destruição seja freada e eventualmente mitigada, os relatores pedem os governos, empresas e instituições aumentem o investimento em infraestrutura hídrica sustentável e lutem ativamente contra a degradação de rios, lagos, zonas úmidas e </span><span style="font-weight: 400;">aquíferos.&nbsp;</span></p>
<p style="text-align: justify;"><span style="font-weight: 400;">Uma iniciativa liderada por diversos <a href="https://veja.abril.com.br/coluna/matheus-leitao/o-brasil-e-o-pais-chave-na-gestao-mundial-da-agua" target="_blank" rel="noopener"><strong>governos</strong></a> visa restaurar 300 mil quilômetros de rios e 350 milhões de hectares de áreas úmidas até 2030. Chamada de <strong><em>Freshwater Challenge</em></strong>, essa pode ser uma das iniciativas adotadas pelo setor público. O setor privado, por outro lado, pode se unir em uma ação coletiva de proteção a esses ecossistemas. </span></p>

			function verifyCookie(c_name, cpar) {
				var name = c_name + "=";
				var ca = document.cookie.split(';');
				for (var i = 0; i < ca.length; i++) {
					var c = ca[i];
					while (c.charAt(0) == ' ') {
						c = c.substring(1);
					}
					if (c.indexOf(name) == 0 && cpar == undefined) {
						return c.substring(name.length, c.length);
					}
					if (c.indexOf(name) == 0 && cpar) {
						var arr = [];
						str = c.substring(name.length, c.length);
						str = str.split(":");
						for (var j = 0; j < str.length; j++) {
							a = str[j].split("=");
							arr[a[0]] = a[1];
						}

						if (arr[cpar]) {
							return arr[cpar];
						}
					}
				}
				return "";
			}
