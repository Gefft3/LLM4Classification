Ela foi resgatada nessa quinta-feira (17).&nbsp;</p>

<p style="box-sizing: border-box; margin-block: 0px 0.9rem; margin-bottom: 16px; font-size: 18px; line-height: 1.45em; color: rgb(0, 0, 0); font-family: -apple-system, BlinkMacSystemFont, &quot;Segoe UI&quot;, Roboto, &quot;Helvetica Neue&quot;, Arial, &quot;Noto Sans&quot;, sans-serif, &quot;Apple Color Emoji&quot;, &quot;Segoe UI Emoji&quot;, &quot;Segoe UI Symbol&quot;, &quot;Noto Color Emoji&quot;;">O crime aconteceu na cidade de&nbsp;<a class="st_tag internal_tag " href="https://midiamax.uol.com.br/municipios/sidrolandia/" rel="tag" style="box-sizing: border-box; background-color: transparent; text-decoration-line: none; color: rgb(0, 123, 193); box-shadow: none; line-height: 1.45em;" title="Últimas Notícias de Sidrolândia">Sidrolândia</a>, a 70 quilômetros de&nbsp;<a class="st_tag internal_tag " href="https://midiamax.uol.com.br/municipios/campo-grande/" rel="tag" style="box-sizing: border-box; background-color: transparent; text-decoration-line: none; color: rgb(0, 123, 193); box-shadow: none; line-height: 1.45em;" title="Últimas Notícias de Campo Grande">Campo Grande</a>. A mulher contou que o companheiro não aceitava a separação, sendo que a manteve em cárcere desde o dia 16 a torturando. De acordo com ela, o autor usou um frasco de desodorante e um isqueiro para criar um lança-chamas afirmando que colocaria fogo nela.</p>

<p style="box-sizing: border-box; margin-block: 0px 0.9rem; margin-bottom: 16px; font-size: 18px; line-height: 1.45em; color: rgb(0, 0, 0); font-family: -apple-system, BlinkMacSystemFont, &quot;Segoe UI&quot;, Roboto, &quot;Helvetica Neue&quot;, Arial, &quot;Noto Sans&quot;, sans-serif, &quot;Apple Color Emoji&quot;, &quot;Segoe UI Emoji&quot;, &quot;Segoe UI Symbol&quot;, &quot;Noto Color Emoji&quot;;">Depois, segundo ela, o autor usou um fio desencapado dizendo que iria eletrocutá-la. A vítima ainda contou que o companheiro usou um lençol para enforcá-la e a ameaçou com uma&nbsp;<a class="st_tag internal_tag " href="https://midiamax.uol.com.br/tag/faca/" rel="tag" style="box-sizing: border-box; background-color: transparent; text-decoration-line: none; color: rgb(0, 123, 193); box-shadow: none; line-height: 1.45em;" title="Notícias sobre faca">faca</a>&nbsp;dizendo que iria matá-la.</p>

<p style="box-sizing: border-box; margin-block: 0px 0.9rem; margin-bottom: 16px; font-size: 18px; line-height: 1.45em; color: rgb(0, 0, 0); font-family: -apple-system, BlinkMacSystemFont, &quot;Segoe UI&quot;, Roboto, &quot;Helvetica Neue&quot;, Arial, &quot;Noto Sans&quot;, sans-serif, &quot;Apple Color Emoji&quot;, &quot;Segoe UI Emoji&quot;, &quot;Segoe UI Symbol&quot;, &quot;Noto Color Emoji&quot;;">Para conseguir ajuda, a mulher conseguiu em uma distração do autor pedir socorro para uma amiga que chamou a polícia. Quando os militares chegaram a casa, a vítima saiu pelo portão aos prantos. O homem estava na residência e acabou preso. O casal estava junto há seis meses.</p>

<p style="box-sizing: border-box; margin-block: 0px 0.9rem; margin-bottom: 16px; font-size: 18px; line-height: 1.45em; color: rgb(0, 0, 0); font-family: -apple-system, BlinkMacSystemFont, &quot;Segoe UI&quot;, Roboto, &quot;Helvetica Neue&quot;, Arial, &quot;Noto Sans&quot;, sans-serif, &quot;Apple Color Emoji&quot;, &quot;Segoe UI Emoji&quot;, &quot;Segoe UI Symbol&quot;, &quot;Noto Color Emoji&quot;;">Midiamax</p>
