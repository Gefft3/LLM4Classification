                                <p style="text-align: justify;">REINALDO SILVA</p>
<p style="text-align: justify;">Da Redação</p>
<p style="text-align: justify;">
</p><p style="text-align: justify;">De agosto de 2023 até agora, a Secretaria de Estado da Saúde (Sesa) confirmou 15 mortes por dengue nos municípios do Noroeste do Paraná. Os registros mais recentes foram divulgados nos boletins de 2 e 9 de julho.</p>
<p style="text-align: justify;">São dois pacientes do sexo masculino. O primeiro tinha 75 anos de idade, apresentava doenças associadas, morava em Marilena e morreu no dia 8 de abril. O outro, com 50 anos, também tinha comorbidade, era morador de Santa Isabel do Ivaí e o óbito foi registrado no dia 22 de abril.</p>
<p style="text-align: justify;">Até então eram 13 mortes distribuídas da seguinte forma: seis em Paranavaí, três em Planaltina do Paraná, uma em Amaporã, uma em Guairaçá, uma em Marilena e uma em Santa Isabel do Ivaí.</p>
<figure id="attachment_149128" aria-describedby="caption-attachment-149128" style="width: 395px" class="wp-caption alignleft"><a href="https://diariodonoroeste.com.br/wp-content/uploads/2024/07/Dengue-2.jpg"><img loading="lazy" decoding="async" class="wp-image-149128 " src="https://diariodonoroeste.com.br/wp-content/uploads/2024/07/Dengue-2-300x195.jpg" alt="" width="395" height="257" srcset="https://diariodonoroeste.com.br/wp-content/uploads/2024/07/Dengue-2-300x195.jpg 300w, https://diariodonoroeste.com.br/wp-content/uploads/2024/07/Dengue-2-1024x667.jpg 1024w, https://diariodonoroeste.com.br/wp-content/uploads/2024/07/Dengue-2-768x500.jpg 768w, https://diariodonoroeste.com.br/wp-content/uploads/2024/07/Dengue-2-600x391.jpg 600w, https://diariodonoroeste.com.br/wp-content/uploads/2024/07/Dengue-2-750x488.jpg 750w, https://diariodonoroeste.com.br/wp-content/uploads/2024/07/Dengue-2.jpg 1080w" sizes="(max-width: 395px) 100vw, 395px"></a><figcaption id="caption-attachment-149128" class="wp-caption-text">frio dos últimos dias pode reduzir a circulação do inseto, mas a vigilância quanto aos possíveis criadouros deve ser constante. Foto: Reprodução/internet</figcaption></figure>
<p style="text-align: justify;">Para fins de controle, a Sesa adotou um calendário diferente do convencional, com o período epidemiológico começando entre os últimos dias de julho e os primeiros de agosto – e não em janeiro.</p>
<p style="text-align: justify;">Considerando esse recorte, na temporada 2021-22, o Noroeste do Paraná registrou duas mortes por complicações da dengue. No ciclo seguinte, 2022-23, o número subiu para seis.</p>
<p style="text-align: justify;">Em relação aos casos positivos da doença, a Sesa contabilizou 5.109 em 2021-22 e 5.850 em 2022-23. Para dar uma ideia da gravidade da situação atual, o ano epidemiológico ainda não terminou, mas o Noroeste do Paraná já soma 13.829 confirmações.</p>
<p style="text-align: justify;">
</p><p style="text-align: justify;"><strong>MORTES</strong></p>
<p style="text-align: justify;">A seguir, a lista de óbitos por ordem cronológica dos registros:</p>
<p style="text-align: justify;">3 de agosto – home de 69 anos – Marilena</p>
<p style="text-align: justify;">2 de janeiro – homem de 75 anos – Paranavaí</p>
<p style="text-align: justify;">7 de janeiro – mulher de 44 anos – Planaltina do Paraná</p>
<p style="text-align: justify;">7 de janeiro – mulher de 50 anos – Planaltina do Paraná</p>
<p style="text-align: justify;">27 de janeiro – mulher de 92 anos – Paranavaí</p>
<p style="text-align: justify;">1º de fevereiro – homem de 71 anos – Paranavaí</p>
<p style="text-align: justify;">4 de fevereiro – mulher de 60 anos – Paranavaí</p>
<p style="text-align: justify;">4 de fevereiro – homem de 83 anos – Planaltina do Paraná</p>
<p style="text-align: justify;">3 de março – homem de 81 anos – Amaporã</p>
<p style="text-align: justify;">6 de abril – homem de 93 anos – Paranavaí</p>
<p style="text-align: justify;">6 de abril – mulher de 83 anos – Guairaçá</p>

<p style="text-align: justify;">11 de abril – homem de 69 anos – Paranavaí</p>
<p style="text-align: justify;">22 de abril – mulher de 63 anos – Santa Isabel do Ivaí</p>
<p style="text-align: justify;">22 de abril – homem de 50 anos – Santa Isabel do Ivaí.</p>
<p style="text-align: justify;">
</p><p style="text-align: justify;"><strong>CUIDADOS</strong></p>
<p style="text-align: justify;">Algumas condições favorecem a proliferação do Aedes aegypti, mosquito transmissor da dengue, por exemplo, a combinação de temperaturas altas e chuva acelera o ciclo de reprodução.</p>
<p style="text-align: justify;">O frio dos últimos dias pode reduzir a circulação do inseto, mas a vigilância quanto aos possíveis criadouros deve ser constante.</p>
<p style="text-align: justify;">É preciso tomar alguns cuidados, sendo o principal a limpeza dos quintais residenciais e das empresas. O descarte correto do lixo é outra prática fundamental nesse sentido. Outra orientação é vedar caixas d’água, cisternas, vasos sanitários em desuso e ralos de cozinhas, banheiros e quintais.</p>

// Esconder btn grupo whats
function hideGrupoWhats() {
  document.querySelectorAll('.alerts').forEach(element => {
    element.style.display = 'none';
    localStorage.setItem('alertClosedTime', Date.now());
    localStorage.setItem('alertClosed', true);
  });
}
// btn fechar
document.addEventListener('DOMContentLoaded', function() {
  document.querySelectorAll('.alert-close').forEach(element => {
    element.addEventListener('click', hideGrupoWhats);
  });
// verifica se foi clicado
  if (checkAlertClosed()) {
    document.querySelectorAll('.alerts').forEach(element => {
      element.style.display = 'none';
    });
  }
});
// se clicado aguarda 5min para deixar carregar novamente
function checkAlertClosed() {
  const alertClosedTime = localStorage.getItem('alertClosedTime');
  const alertClosed = localStorage.getItem('alertClosed');

  if (alertClosed === 'true' && alertClosedTime) {
    const timeDiff = Date.now() - parseInt(alertClosedTime);
    if (timeDiff < 5 * 60 * 1000) {
      return true;
    } else {
      localStorage.removeItem('alertClosed');
      localStorage.removeItem('alertClosedTime');
    }
  }

  return false;
}
</script>

// MODAL AGENDA DN
jQuery(document).ready(function($) {
    var modal = $('#modalMaisInformacoesAgendaDN');

    $('.close-modal-agenda-dn').on('click', function() {
        modal.hide();
    });

    $(window).on('click', function(event) {
        if ($(event.target).is(modal)) {
            modal.hide();
        }
    });
});

function abrir_modal_mais_informacoes_agenda_dn(post_id) {
    
    var texto_btn = '';
    var btn_mais_informacoes = jQuery('.btn_mais_info_'+post_id);
    
    texto_btn = btn_mais_informacoes.html();
    btn_mais_informacoes.html('Carregando...');
    
    jQuery.ajax({
        url: 'https://diariodonoroeste.com.br/wp-admin/admin-ajax.php?action=content_agenda_dn',
        method: 'POST',
        data: {
            evento_id : post_id,
        },
        success: function(retorno) {
            jQuery('#modalMaisInformacoesAgendaDN .modal-body-agenda-dn').html(retorno);
            jQuery('#modalMaisInformacoesAgendaDN').show();
            
            btn_mais_informacoes.html(texto_btn);
        },
        error: function() {
            console.log('erro ao carregar mais informacoes da agenda dn');
        }
    });
}

/* Modal Load */
function openLoadingModal() {
    var modal = document.getElementById('loadingModal');
    modal.style.display = 'block';

    // Fechar automaticamente após 3 segundos (3000 milissegundos)
    setTimeout(function() {
        closeLoadingModal();
    }, 3200); // Altere o valor conforme necessário
}

// Função para fechar o modal
function closeLoadingModal() {
    var modal = document.getElementById('loadingModal');
    modal.style.display = 'none';
}
